
- Need to add stuff to Visual Studio to support Edge browser as a component
Install Microsoft.Web.WebView2 from the Tools->NuGet Package Manager->Manage NuGet Packages for Solution, then click on Browse and find it.
More info here:  https://docs.microsoft.com/en-us/microsoft-edge/webview2/get-started/wpf
